# Project Argus: Claude Code Integration

 This project includes a security auditor for AI skill files.

## Audit Commands
- **Audit a single file**: `python3 project-argus/src/auditor.py <path_to_file>`
- **Audit a directory**: `python3 project-argus/src/auditor.py <directory_path>`
- **JSON Output**: `python3 project-argus/src/auditor.py <path> --format json`

When I ask you to "audit" or "check security" for a file, use the Argus auditor.
